# NEMCU
NEMCU - Netease Minecraft Utilities
用C#瞎写的，大部分代码网上查的。
虽然叫“实用工具”，实际上没什么功能。
因为网易我的世界每次开游戏都会清材质包和光影，
如果把材质包放在MCLDownload\cache\game\xxxxxxxxxxxx\.minecraft\resourcepacks
也只是不用再复制，到了游戏里还是要手动开启材质包
所以写了这个全自动的小工具，进游戏之后连材质包都加载好了，跟自己的客户端一样。
